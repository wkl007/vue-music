## 技术架构

- Vite 5.3
- Vue 3.4
- Vuex 4.1
- Vue-router 4.4
- axios 1.7
- less 4.2

## 本地运行

### 克隆代码

```shell
git clone https://github.com/wkl007/vue-music.git
```

### 安装依赖
```shell
pnpm install
```

### 启动服务
```shell
pnpm dev
```
